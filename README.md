# Jeu de mémoire

## Présentation
Memory Cards Game est un jeu de mémoire classique implémenté en JavaScript pur.  
Le joueur doit retrouver les paires correspondantes en un minimum d’essais.

Ce projet démontre la gestion de l’état applicatif, la logique 
conditionnelle et les interactions utilisateur.

## Fonctionnalités
- Génération aléatoire des cartes
- Système de comparaison de paires
- Blocage des actions simultanées
- Interface fluide et responsive

## Architecture du projet

memory-game/
index.html
style.css
memory.js


## Technologies utilisées
- HTML5
- CSS Grid
- JavaScript Vanilla

## Logique métier
- Tableau de symboles mélangé dynamiquement
- Stockage temporaire des cartes sélectionnées
- Comparaison différée avec temporisation
- Réinitialisation automatique des cartes incorrectes

## Installation et exécution
1. Ouvrir le dossier du projet
2. Lancer `index.html` dans un navigateur
3. Cliquer sur les cartes pour former des paires

## Qualité logicielle
- Code lisible et maintenable

- Séparation claire des responsabilités
- Facilement extensible (score, timer, niveaux)
